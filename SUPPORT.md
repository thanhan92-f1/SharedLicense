# Support

## What this project provides

This repository provides a HostBill provisioning module integrating with the SharedLicense Reseller API.

## Support scope

Supported:

- Bug reports with clear reproduction steps.
- Documentation improvements.
- Reasonable compatibility fixes for HostBill environments.

Out of scope (may be declined):

- HostBill core issues unrelated to this module.
- Custom feature work without a clear spec.
- Troubleshooting that requires access to your production secrets or systems.

## How to request help

When opening an issue, include (sanitize sensitive data):

- Module version (see `SharedLicense::$version`).
- HostBill version.
- PHP version.
- Exact error message(s).
- Whether debug logging is enabled.
